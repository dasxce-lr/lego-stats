Set Value (New/Sealed)

This chart represents the current new/sealed value of the set Brick Bank since November 2018. Very high accuracy
This is the fourth corner building to be released, so we only need one more normal street edge building to make a fully connected square. With the exception of the first year we saw these Modular Buildings (2007), there has been one building released per year. There are five minifigs, all with the plain face as is customary for these modular building sets.

It was retired in November 2018 with a lifespan of 34 months. The current value for a new and sealed Brick Bank is estimated at $569 with an average yearly gain of about 13% which is higher than other Modular Buildings sets. On the secondary markets, the typical price range for the set MISB/NISB is between $551 and $700. The current used prices for Brick Bank can be found in the range of $420 and $550 depending on the set's condition, with an average price of $458 if it is in good overall condition.

About Icons Modular Buildings

The LEGO ICONS Modular Buildings subtheme is an extension of the Advanced Models theme. Modular Buildings features large, detailed, connectable buildings, aimed at adults and teenagers.

Modular Buildings currently consists of 10 sets that as of now has run from 2013 to 2022. As of today, 10251 Brick Bank is the most valuable Modular Buildings set with an estimated new/factory sealed value of $569.
Summary
Icons Modular Buildings Brick Bank set 10251 was released on January 2, 2016, retailing for $170, it has since been retired. New, unopened sets are valued at $569, used ones can be found for $458.
Read more
Make a secure deposit at the highly respected Brick Bank, featuring an array of intricate details and hidden surprises. Easy-to-remove building sections provide access to the detailed interior, comprising a bank with an atrium foyer, tiled floor, arched windows, ornate chandelier, lockable vault and a transaction counter with security glass; a laundromat with printed window, tiled floor and 4 laundry machines; plus 2 second-floor offices with an array of detailed furniture, fixtures and accessories. The exterior of the building features a detailed sidewalk and an elaborate façade with carving and statue décor, decorative roofline, large arched windows, central balcony, clock and an accessible roof terrace featuring a large skylight.
